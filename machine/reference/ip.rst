=======================================
ip
=======================================

.. Get the IP address of one or more machines.

１つまたは複数マシンの IP アドレスを表示します。

.. code-block:: bash

   $ docker-machine ip dev
   192.168.99.104
   $ docker-machine ip dev dev2
   192.168.99.104
   192.168.99.105
