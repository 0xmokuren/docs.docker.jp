========================================
toolbox
========================================

.. Available for both Windows and Mac, Docker Toolbox installs the Docker Client, Machine, Compose, and Kitematic.

Windows と Mac の両方で利用可能な Docker Toolbox は、Docker クライアント、Machine、Compose、Kitematic をインストールします。

.. Learn about Docker Toolbox. See Docker Toolbox Overview for a quick tour of how to install Toolbox, and get started with Docker Machine, Kitematic, and Docker Compose.

**Docker Toolbox の詳細を学ぶ** ： Toolbox のインストールの仕方や簡単な使い方を知るには :doc:`overview` をご覧ください。そして Docker Machine 、 Kitematic 、 Docker Compose を使っていきましょう。

.. Ready to download Toolbox? Download Docker Toolbox from products.

**Toolbox をダウンロード済みですか？** ： プロダクトのページから `Docker Toolbox をダウンロード <https://www.docker.com/products/docker-toolbox>`_ します。
