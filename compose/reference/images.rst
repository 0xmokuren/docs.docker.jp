=======================================
docker-compose images
=======================================

.. code-block:: bash

   使い方: docker-compose images [オプション] [サービス...]

   オプション:
       -q, --quiet  ID のみを表示

.. List images used by the created containers.

作成済みのコンテナが使用しているイメージ一覧を表示します。
