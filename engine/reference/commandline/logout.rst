.. logout

=======================================
logout
=======================================

.. code-block:: bash

   Usage: docker logout [SERVER]
   
   Log out from a Docker registry, if no server is
   specified "https://index.docker.io/v1/" is the default.
   
     --help          Print usage

.. For example:

例：

.. code-block:: bash

   $ docker logout localhost:8080
